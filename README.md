# Cloud Architecture for Agentic AI

Repository di supporto al project work di Diego Tomassone per il Master in Data Science e AI.

## Descrizione

Questo repository raccoglie materiali a supporto del project work dedicato alla progettazione di un’architettura cloud per sistemi di AI agentica, con particolare attenzione a:

- architetture multilivello (IaaS, PaaS, SaaS);
- orchestrazione di servizi e workflow;
- sicurezza, sandboxing e Zero Trust;
- integrazione con agenti intelligenti e sistemi multiagente.

L’obiettivo è fornire una vista sintetica ma operativa delle scelte architetturali e dei principi di progettazione.

## Contenuti

- `diagram/architettura-cloud-agentica.png`  
  Schema architetturale multilivello dell’ambiente cloud per AI agentica.

- `workflow/workflow-agentico.md`  
  Esempio di workflow agentico con interazione tra LLM, orchestratore, servizi cloud e sandbox di esecuzione.

- `notes/security-and-governance.md`  
  Note sintetiche su sicurezza, IAM, Zero Trust, audit e monitoraggio.

- `docs/project-work.pdf`  
  Versione PDF del project work “Cloud Architecture for Agentic AI”.

## Tecnologie e concetti chiave

- Cloud computing (AWS/Azure/GCP – modello concettuale)
- Container, orchestrazione e microservizi
- Zero Trust Architecture e IAM
- AI agentica e sistemi multiagente
- Logging distribuito, osservabilità e resilienza

## Possibili estensioni future

- Esempi di configurazione infrastrutturale (Terraform, Helm, ecc.)
- Integrazione con un orchestratore di agenti
- Prototipi di policy di sicurezza per ambienti agentici
